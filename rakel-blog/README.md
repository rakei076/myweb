
# 🌟 Rakel 个人博客网站

一个基于 React + TypeScript 构建的现代化个人博客网站，采用独特的活页夹设计风格，专注于个人介绍与自我记录。

## 🗒️ 协作历史摘要

- 主题从蓝白全面升级为灰白极简配色，左侧装订栏渐层与材质同步细化。
- 兴趣页重构为全屏海报轮播，去除熟练度，仅保留标题并支持自定义背景图。
- 旅游页的 3D 地球降低自转速度并关闭默认弹窗，交互更加舒适。
- 新增文章页与留言板：文章与成长时间轴联动；留言区升级为贴吧式楼层结构。
- README 与数据结构更新，指导如何管理背景素材、文章封面与留言内容。

## ✨ 特色功能


### 🎨 设计亮点
- **活页夹样式布局** - 左侧装饰 | 中间内容展示 | 右侧页面导航
- **灰白极简配色** - 参考高级网页配色设计，主色为灰白，辅以淡色渐变
- **流畅动画效果** - 基于 Framer Motion 的页面切换和交互动画
- **响应式设计** - 完美适配桌面、平板和移动端

### 📑 页面功能

#### 🙋‍♂️ 自我介绍页
- 个人基本信息展示
- 联系方式（邮箱、微信、GitHub）
- 个人简介和学习背景


#### ❤️ 兴趣爱好页
- 游戏、技术、生活三大类兴趣极简展示
- **全屏无边框海报**，仅保留兴趣标题，背景可自定义
- 左右滑动切换，支持键盘/触屏

#### 💼 作品展示页
- **3D卡片翻转动画** - 点击卡片查看详细信息
- GitHub 仓库直连和在线预览
- 技术栈标签和项目分类


#### 🗺️ 旅游经历页
- **3D可旋转世界地图** - 基于 Three.js 构建
- 交互式地点标记和详情展示
- 时间轴整合和照片展示

#### ⚡ 技术栈页
- **技能雷达图** - SVG绘制的可视化技能展示
- 分类技能列表和熟练度展示
- 技能关联项目展示


#### ⏰ 成长时间轴页
- 垂直时间轴设计，支持文章与成长事件混排
- 教育、项目、成就、里程碑、**文章**分类
- 文章节点可一键跳转到对应内容
- 滚动触发动画效果


#### 🔖 友链书签页
- 分类卡片式布局（技术、工具、学习、娱乐）
- 搜索功能和实时筛选
- 外链直跳和图标展示
#### 💬 留言板（贴吧楼层风格）
- 支持楼层编号、头像首字母、楼主标识
- 回复区与上下楼连接线，体验类似贴吧/论坛
- 支持表单留言，自动刷新展示

## 🛠️ 技术栈

### 前端框架
- **React 18** - 现代化前端框架
- **TypeScript** - 类型安全的JavaScript
- **Vite** - 快速构建工具

### 样式和动画
- **Styled Components** - CSS-in-JS 样式解决方案
- **Framer Motion** - 强大的动画库
- **响应式设计** - 移动端优先的布局

### 3D和可视化
- **Three.js** - 3D图形库
- **@react-three/fiber** - React Three.js 渲染器
- **@react-three/drei** - Three.js 工具集
- **SVG** - 自定义雷达图和图标

### 路由和工具
- **React Router** - 单页应用路由
- **Lucide React** - 现代化图标库
- **React Intersection Observer** - 滚动动画触发

## 🎯 设计理念与近期更新

### 活页夹概念
整个网站采用活页夹的设计理念，模拟真实的个人档案夹：
- **左侧装饰栏** - 活页夹的装订边，带有装饰性圆点动画
- **中间内容区** - 活页夹的页面内容，带有打孔效果
- **右侧导航栏** - 页面标签，方便快速跳转


### 视觉风格
- **极简灰白** - 参考高级网页配色书籍，主色为灰白，辅以淡色渐变
- **全屏兴趣海报** - 兴趣页仅展示大标题与背景
- **楼层留言板** - 贴吧/论坛式互动体验
- **文章与时间轴联动** - 文章可作为成长节点展示
- **流畅动画** - 页面切换和交互都有平滑的动画效果
- **响应式** - 在不同设备上都有良好的体验

## 🔧 自定义配置

### 修改主题颜色
编辑 `src/styles/theme.ts` 文件来自定义颜色方案。

### 添加新页面
1. 在 `src/components/Pages/` 创建新的页面组件
2. 在 `src/data/navigation.ts` 添加导航项
3. 在 `src/App.tsx` 添加路由配置

### 更新个人信息
各页面的数据存储在 `src/data/` 目录下，可根据需要修改：
- 个人信息：直接在组件中修改
- 兴趣爱好：`interests.ts`
- 项目展示：`projects.ts`
- 旅游经历：`travel.ts`
- 技能数据：`skills.ts`
- 时间轴：`timeline.ts`
- 书签链接：`bookmarks.ts`

## 📱 响应式支持

网站在以下设备上经过优化：
- **桌面端** (1200px+) - 完整的三栏布局
- **平板端** (768px-1199px) - 自适应布局调整
- **移动端** (<768px) - 单栏布局，导航栏折叠
